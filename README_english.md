[SIMANFOR](https://www.simanfor.es) is a Decision Support System that allows the simulation of forest management alternatives.

This repository is focused on understanding the results obtained from simulations and how to interpret and analyze them to get the most out of them.

---

## 📜 Content

* 💻 **analisis_resultados**: Folder containing an R script prepared to create some graphs showing the evolution of the forest stand and a comparison of the accumulated wood volume (including harvested) under different silvicultural scenarios. You can view the results of this analysis in the *.html* file located in the same folder.
  * 💡 If you already have more advanced knowledge of R and are looking to create a more professional report, then use this report template:
    - 🇪🇸 [Spanish report template](./informes_simulaciones/plantilla_informe_espanol/)
    - 🇬🇧 [English report template](./informes_simulaciones/report_template_english/)
* 📚 **informes_simulaciones**: examples of graphical reports generated from simulations performed with SIMANFOR.
* 📊 **Ppinaster-Del_Rio_2006**: Results obtained for the 5 site qualities of *Pinus pinaster* in Castilla y León proposed by [Del Río et al., 2006](https://www.researchgate.net/publication/265520003_Manual_de_gestion_para_masas_procedentes_de_repoblacion_de_Pinus_pinaster_Ait_Pinus_sylvestris_L_y_Pinus_nigra_Arn_en_Castilla_y_Leon)
* 📊 **Ps-no_maderables**: Results obtained for *Pinus sylvestris* plots in Catalonia obtained from the [National Forest Inventory](https://www.miteco.gob.es/es/biodiversidad/temas/inventarios-nacionales/inventario-forestal-nacional/default.aspx).
* 💾 **ejemplo_resultado.xlsx**: Example file to see the type of results that can be obtained from SIMANFOR
* 📖 🇪🇸 **SIMANFOR_resultados.pdf**: Explanatory document for each section of the SIMANFOR results files in Spanish
* 📖 🇬🇧 **english/SIMANFOR_results.pdf**: Explanatory document for each section of the SIMANFOR results files in English
